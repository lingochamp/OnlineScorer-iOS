EZOnlineScorerRecorderError

EZOnlineScorerRecorder error type

  • EZOnlineScorerRecorderErrorRecorderError: EZOnlineScorerRecorder fail to record. See NSUnderlyingErrorKey in userInfo for detail.
  • EZOnlineScorerRecorderErrorConnectionError: EZOnlineScorerRecorder fail to communicate with server. See NSUnderlyingErrorKey in userInfo for detail, the underlying error is a EZAudioReaderError.
  • EZOnlineScorerRecorderErrorResponseJSONError: EZOnlineScorerRecorder fail to decode response. Please contact developer if this error occored, the underlying error is a EZAudioSocketError.
  • EZOnlineScorerRecorderErrorInvalidParameter: EZOnlineScorerRecorder input parameters invalid. Please make sure you’re using a valid EZOnlineScorerRecorderPayload.
